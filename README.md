# NumPy Training – Fundamentals and Applications

## 📘 Overview
This repository contains materials from a **NumPy training session** delivered at **EMJE**, a student club at the National School of Applied Sciences (ENSA).  
The course introduces students to **NumPy**, the core Python library for numerical computing, and provides hands-on practice with arrays, matrix operations, and data manipulation.

---

## 🎯 Objectives
- Build a strong foundation in numerical computing with Python.  
- Learn how to create, manipulate, and operate on arrays.  
- Understand broadcasting, indexing, and slicing.  
- Apply NumPy functions in data science and engineering contexts.  
- Provide reproducible examples for self-study and future projects.  

---

## 📂 Contents
- **Notebooks**: Step-by-step explanations with code samples.  
- **Exercises**: Practical problems to reinforce learning.  
- **Examples**: Applications in data analysis and machine learning.  

---
